## ğŸ“ Field Validation Rules

### Required Fields
- HTML5 `required` attribute
- Browser native validation
- Cannot submit with empty required fields

### Platform Checkboxes
- Must select at least one
- Validated in backend
- Returns 400 if none selected

### Message Fields
- Character limits enforced
- Live character counter
- maxLength attribute prevents overflow

### URL Fields (Portfolio)
- type="url" for validation
- Optional, so can be left empty
